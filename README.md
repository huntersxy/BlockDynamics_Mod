# BlockDynamics 模组说明

## 概述
BlockDynamics 是一个基于 Minecraft Forge 的模组。
当前功能：

-  通过监测区块内生物数量来控制生物繁殖，避免特定区域内生物过度密集，提升游戏体验。
- 新增一个方块-实体冻结器，用于通过红石控制冻结指定区域（7*7）内的生物。

## 核心功能

### 繁殖限制
- **监测范围**：以生物父母所在位置为中心的 8x8x8 区域（从坐标 X-8, Y-8, Z-8 到 X+8, Y+8, Z+8）
- **繁殖限制**：当区域内 `Mob` 类实体数量超过配置的maxMobsInChunk值时，繁殖行为将被取消
### 实体冻结器
- **监测范围**：以实体所在位置为中心的 7x7x7 区域（从坐标 X-3, Y-3, Z-3 到 X+3, Y+3, Z+3）
- **冻结条件**：放置时/受到充能时，实体将冻结。取消充能后，实体解冻。

注：当前该方块没有创造物品栏，请使用give 命令获取，方块id为 givetag_block .

## 安装指南

### 玩家安装步骤
1. 确保已安装对应版本的 Minecraft Forge（支持 1.20.1 版本）
2. 将模组的 JAR 文件放入 Minecraft 游戏目录下的 `mods` 文件夹
3. 启动游戏即可加载模组

## 配置说明

### 配置文件位置
```
config/blockd-common.toml
```


### 配置方法
1. 启动一次游戏生成配置文件
2. 编辑配置文件中的 [maxMobsInChunk](file://E:\mcmod\forge-1.20.1-47.4.0-mdk\src\main\java\com\huntersxy\blockd\Config.java#L25-L25) 值，设置允许的最大生物数量，默认为10
3. 保存文件并重启游戏，配置将生效
